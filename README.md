# Otopark Yönetim Sistemi
Bu proje, bağlı liste veri yapısı kullanılarak geliştirilmiş bir otopark yönetim simülasyonudur.

## Özellikler
* Araç girişi ve plaka kontrolü.
* Boşluklu plaka desteği (Örn: 34 ABC 123).
* Bağlı liste ile dinamik araç yönetimi.
* Zaman bazlı ücret hesaplama.
* Plakaya göre araç arama ve konum tespiti.
